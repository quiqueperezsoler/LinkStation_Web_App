 Link Stations Web Application
This program solves the most suitable link station for any given point in the 2D space. It takes into account the reach and power of the link stations which is given in the form [x, y, r] with x = position in the X-axis, y= position in the Y-axis, r = reach.

The Link Stations are located as follows:

Link Station A = [0, 0, 10]

Link Station B = [20, 20, 5]

Link Station C = [10, 0, 12]
